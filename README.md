# ai-cafe-reviews

### Тема
Система позволяет клиентам оставлять отзывы о кафе и помогает анализировать их.
1. Сбор отзывов — возможность оставлять текстовые или голосовые отзывы.
2. Анализ отзывов — определение тональности (положительный, отрицательный) и ключевых слов.
3. Отчёты — предоставление статистики о среднем уровне удовлетворённости клиентов.
4. Агенты для задач — агент для обработки отзывов, включая подсчёт количества положительных и отрицательных отзывов.
5. Уведомления — оповещение менеджеров о негативных отзывах.

### Оценивание
1. Сбор отзывов (2 балла): система корректно принимает и сохраняет отзывы.

2. Анализ отзывов (2 балла): анализ тональности выполняется корректно.

3. Отчёты (1 балл): статистика по отзывам понятна и полезна.

4. Агенты (1 балл): агент выполняет подсчёт отзывов и анализ ключевых слов.

5. Наличие интерфейса (1 балл): реализован Telegram-бот или Streamlit и тд.
